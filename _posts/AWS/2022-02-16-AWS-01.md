## EC2? 🍪

<center>  EC2 : Elastic Compute Cloud 컴퓨터 임대 서비스이다.</center>
{: .notice--danger}

1. 지역 정하기

2. 인스턴스 시작 ( 컴퓨터 한대한대를 인스턴스라고 한다. )

3. 인스턴스 세부정보 구성

- 인스턴스 갯수가 1개인것을 확인할것

4. 키페어 생성

인바운드 규칙 : 외부에서 EC2로 접속 ( 최소한으로 열어놓는다 )

아웃바인드 규칙 : EC2에서 외부로 접속 ( 최대한 열어놓는다 )
